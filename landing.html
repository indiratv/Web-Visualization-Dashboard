Summary

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, data from the OpenWeatherMap API was used to assemble a dataset on over 500 cities.

After assembling the dataset, the Matplotlib library was used to plot the various aspects of the weather vs latitude. Analysis was done to see the how the following parameters change by latitude - temperature, cloudiness, wind speed and humidity.This site provides the source data and visualizations created as part of the analysis as well as explanations and descriptions of any trend and correlations witnessed.
